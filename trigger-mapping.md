## Scoring-Regeln

### Regel 1 — We Are / We Are Not Adjektive (Skill 03)
Jedes Adjektiv auf CORE gibt +2 Punkte auf den zugeordneten Trigger.
Jedes Adjektiv auf ASPIRATION gibt +1 Punkt.
Jedes Adjektiv auf NICHT gibt -1 Punkt (aktive Abgrenzung).
Jedes Adjektiv auf UNSICHER gibt 0 Punkte.

### Regel 2 — Fascinate Advantage (Skill 02, Methode A)
Primary Advantage: +4 Punkte auf den gemappten Trigger (siehe Tabelle unten)
Secondary Advantage: +2 Punkte auf den gemappten Trigger
Dormant Advantage: -1 Punkt auf den gemappten Trigger

### Regel 3 — Brand Archetype (Skill 02, Methode B)
Primary Archetype: +3 Punkte auf den gemappten Trigger (siehe Tabelle unten)
Shadow Archetype: +1 Punkt auf den gemappten Trigger

### Regel 4 — Trigger-Hierarchie
Primary Trigger = hoechster Gesamtscore
Secondary Trigger = zweithochster Score
Dormant Trigger = niedrigster Score
Bei Gleichstand: Fascinate Advantage hat Prioritaet (Regel 2 > Regel 1)

- warm: PA +2
- bold: IN +2, PO +2
- innovative: IN +2
- unique: IN +2, MY +2

NICHT-Woerter (angenommen): authoritative, formal, glamorous
- authoritative: PO -1
- formal: AL -1, PR -1
- glamorous: PR -1, PA -1

TOTAL:
PA: 4 + 3 + 2 + 2 + 2 - 1 = 12
IN: 2 + 1 + 2 + 2 + 2 + 2 = 11
PO: 2 - 1 = 1
PR: -1 - 1 = -2
TR: 0
MY: 2
AL: -1
